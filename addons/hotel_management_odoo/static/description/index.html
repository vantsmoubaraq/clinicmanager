Overview

Hotel Management Module helps us to run our hotel smoothly by handling all staffs such as Manager, receptionist, maintenance manager,maintenance worker, etc. Using this module we can view events and add viewers to it and hence we can attract the customers .And also we have vehicle management to attract the travelers to explore the view points.

Features

Room Reservation

Reserve Hotel Rooms based on requirement of Room Capacity and Room Type

Service, Food & Vehicle Management

Order Service, Food and vehicle as per need

Event Management

Book Slots for your favorite Event

Maintenance Management

Request for Maintenance work as per need

Add Services.

To add Services, select the Need Service checkbox. Then, under the Service tab, the services can be added.

Add Vehicle.

To add vehicles, select the Need Vehicle checkbox. Then, under the Fleet tab, the vehicles can be added.

Add Food.

To add Food, select the Need Food checkbox. Then, under the Food tab, the Food items can be added.

Add Events.

To add Events, select the Need Event checkbox. Then, under the Events tab, the Events can be added.

To check out of the room, click CHECK-OUT.

For creating the Invoice, click CREATE INVOICE

Click SET AS DONE for completing the Checkout procedure.
